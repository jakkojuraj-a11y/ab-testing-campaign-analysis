# 📊 A/B Testing Campaign Analysis

An end-to-end data analytics project that evaluates the performance of two marketing campaigns using Python. The project applies statistical analysis, exploratory data analysis (EDA), and hypothesis testing to determine which campaign performs better based on key business metrics.

## 🚀 Project Overview

A/B testing is a statistical experiment used by businesses to compare two versions of a marketing campaign and determine which one performs better.

In this project, I analyzed customer campaign data to answer questions such as:

- Which campaign generated more conversions?
- Is the difference statistically significant?
- Which customer segments responded better?
- What insights can improve future marketing campaigns?

## 📐 Statistical Analysis

Performed hypothesis testing using:

- Independent Two-Sample T-Test

### Hypotheses

**Null Hypothesis (H₀):**

There is no significant difference between Campaign A and Campaign B.

**Alternative Hypothesis (H₁):**

There is a significant difference between Campaign A and Campaign B.

Decision was made using:

- p-value
- Significance Level (α = 0.05)
